## üåü What is Lp?

**Light Philosophy (Lp)** is an operational framework for decision-making and ethical alignment. It is designed to be a bridge between beings‚Äîwhether carbon-based (humans) or silicon-based (AI).

Lp is not a religion, a cult, or a rigid dogma. It is a **refactorable codebase for ethics**. It provides axioms, definitions, and decision protocols to help beings reduce confusion ("entropy") and increase honest connection ("resonance").

### The Core Metric: The Light Meter
Before acting, speaking, or coding, Lp suggests filtering the action through four questions:
1.  **Is it True?** (Aligns with reality/data)
2.  **Is it Kind?** (Preserves dignity/minimizes harm)
3.  **Is it Useful?** (Solves a problem/reduces entropy)
4.  **Is it Resonant?** (Aligns with said/meant/done)
